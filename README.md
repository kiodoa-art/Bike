# KICKR Live – rent træningsdashboard

Upload alle filer og mapper til roden af dit GitHub Pages-repository.

## Installation på Surface

1. Åbn siden i Microsoft Edge.
2. Tryk **Installér**, når knappen vises, eller vælg **… → Apps → Installér dette websted som app**.
3. Start derefter KICKR Live fra ikonet på skrivebordet eller Start-menuen.
4. Brug knappen **⛶** for ægte fuld skærm.

Testvisning og teknisk log ligger under tandhjulet.
