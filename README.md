# KICKR Live – Studio dashboard

Upload alle filer og mapper til roden af dit GitHub Pages-repository.

## Det nye dashboard

- Stort live watt-tal med 3-sekunders udjævning i sidefeltet.
- Live sammenligning af turens gennemsnit mod sidste tur og 30-dages-snittet.
- Referenceværdier gemmes lokalt via tandhjulet.
- Effektgraf for de seneste 10 minutter.
- Selvstændig Bluetooth-forbindelse til en standard pulsmåler, fx iGPSPORT HR70.
- Spotify-visning, fullscreen og PWA-installation er bevaret.

## Installation på Surface

1. Åbn siden i Microsoft Edge.
2. Tryk **Installér**, når knappen vises, eller vælg **… → Apps → Installér dette websted som app**.
3. Start derefter KICKR Live fra ikonet på skrivebordet eller Start-menuen.
4. Brug knappen **⛶** for ægte fuld skærm.

## Før første træning

Åbn tandhjulet og indtast gennemsnitswatt fra din sidste tur samt dit 30-dages-snit. Tallene gemmes kun i browseren på den enhed, hvor appen bruges.

Testvisning og teknisk log ligger også under tandhjulet.
