# KICKR Live – Spotify-styring

Upload disse fire filer til roden af Bike-repositoriet og erstat de eksisterende filer:

- index.html
- sw.js
- spotify-controls.js
- spotify-controls.css

Efter GitHub Pages er færdig:

1. Luk den installerede KICKR Live-app helt.
2. Åbn siden i Edge og tryk Ctrl+F5.
3. Åbn appen igen.
4. Hvis Spotify allerede var forbundet, tryk "Godkend styring" én gang.
5. Log ind hos Spotify og godkend den ekstra tilladelse.

Knapper:
- Forrige
- Start/fortsæt
- Pause
- Stop (pause og gå tilbage til starten)
- Næste
- Lydstyrke med skyder og +/- 5 %

Bemærk: Spotify skal have en aktiv afspilningsenhed. Nogle Spotify Connect-enheder tillader ikke fjernstyring af lydstyrken.
